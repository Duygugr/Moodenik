# 🎵 Moodenik

Moodenik, kullanıcının ruh haline ve tercih ettiği dile göre müzik önerileri sunan modern bir web uygulamasıdır. Kullanıcılar ruh hallerine uygun Spotify çalma listelerini kolayca keşfedebilir ve dinleyebilir.

## 🚀 Özellikler

- 🎭 Ruh haline göre müzik önerileri
- 🌍 Dil seçimi
- 🎵 Spotify çalma listesi entegrasyonu
- 📱 Responsive (mobil uyumlu) tasarım
- ⚡ Hızlı ve kullanıcı dostu arayüz
- 🎨 Modern ve sade kullanıcı arayüzü

## 🛠️ Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- JSON
- Spotify Embed

## 🎯 Nasıl Çalışır?

1. Ruh halinizi seçin.
2. Dinlemek istediğiniz dili belirleyin.
3. Size önerilen çalma listelerini görüntüleyin.
4. Spotify oynatıcısı üzerinden müzikleri dinleyin.
